Space Race

by: Jake Morley

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I struggled to find ideas for what type of game to make, so I looked up fun games to code with Java, and I found this game, which looked fairly easy to make. It also seemed like a fun game to play.

2. What I Learned

If you play again after the game is over, the debris never resets, and more debris spawns making the game nearly impossible. When the players would restart the game, it would be in sudden death state automatically, so I fixed that. Also, the sudden death instructions wouldn't work when playing the second time. Finally, when the player pressed space to play again, it would skip over the menu and instructions state.

3. Architecture of My Game

My first class is just a runner, which just sets up the frame and panel. My second class is the game panel, which sets up all the visuals of the game. The game object class sets up the collision boxes, and dimensions of the characters. The player class controls the movement of the players. The debris class controls the debris. The object manager class draws the objects, updates them, checks collisions etc.

4. Technical Implementation

The part of the game I'm most proud of making is the sudden death state of the game. I implemented this by having the game check if the scores are even when the timer goes off, and if they are, the game goes into sudden death. Otherwise, the game goes to the end state. This was actually pretty hard to implement though, because I also had to make the players freeze when the sudden death mode explains the rules.

5. Most Challenging Part

The hardest part about making this game was probably either making the sudden death state, or making the debris. Also, my wifi was really bad when I was making this game, and I would always lag out of the zoom meeting, which would slow me down.

6. Future Fixes and Updates

I would probably add music to the game, and making the visuals look nicer.
